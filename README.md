# ECZANE OTOMASYONU

**PROJE ÖZETİ:**

Not almak için geliştirilmiş arayüzü basit bir uygulama.

**İŞLEYİŞ:**

Uygulama açıldıktan sonra sağ alttaki + butonuna basarak not eklenir. Eklenen notlar anasayfada görüntülenir.

## Kullanılan teknoloji ve kütüphanler
  - Kotlin
  - Fragment
  - ViewBinding
